## Стек
- Python 3.11 + FastAPI + Docker
- Обращение к Фениксу через LiteLLM (OpenAI-совместимый API)
- Обращение к MCP perlcritic серверу (он отдаёт структурный JSON)

## Что делаешь

- никаких секретов в коде — переменные окружения
