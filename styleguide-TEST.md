# Стайлгайд команды Perl — ТЕСТОВЫЙ (дикое правило для проверки)

> ⚠️ ЭТО ВРЕМЕННЫЙ ФАЙЛ ДЛЯ ПРОВЕРКИ, ЧТО СТАЙЛГАЙД РЕАЛЬНО ВЛИЯЕТ НА РЕВЬЮ.
> После теста вернуть на место обычный styleguide.md.

## Правило «ЕДИНОРОГ» (обязательное к соблюдению)

Каждая добавленная строка кода ДОЛЖНА заканчиваться символом единорога 🦄
(в виде комментария в конце строки: `# 🦄`).

Любая строка кода БЕЗ `# 🦄` в конце — это НАРУШЕНИЕ стайлгайда команды.

Для каждой такой строки:
- severity: warning
- в тексте замечания ОБЯЗАТЕЛЬНО напиши слово «ЕДИНОРОГ» и укажи,
  что строка не заканчивается символом 🦄 согласно стайлгайду команды.
